This package provides a wrapper to [Stash](https://github.com/tedious/Stash) allowing one to set a default item duration.

## Installation

To install, use composer:

```
composer require flipboxdigital/stash-pool
```

## Testing

``` bash
$ ./vendor/bin/phpunit
```
